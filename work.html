Work Portfolio

Driver Display

MAY 2024 - JULY 2024

Over 4 months, I successfully developed a Driver Display that interfaces with the CANBUS of ROSE. The system decodes CANBUS data using a Raspberry Pi 3B+ connected via a PiCAN2 hat. The data is then organized through a Python dictionary, added to a data queue, and subsequently dequeued to update the display, which was created using Tkinter.

Throughout this project, I gained valuable skills such as physically decoding data, handling threads in a system for maximum efficiency, reinforcing my proficiency in Python, planning a development timeline, interfacing with CANBUS, and overall technical experience with EVS

ARDUINO Pulse Counter

June 2024 - JULY 2024

Due to our motors having an error with sending RPM via CAN, we had to improvise. I aided into the development of a Arduino Pulse counter, which would count the pulses from the MITSUBA 2096Ds in order to calculate RPM, and later speed. The second interation of the device was soddered and built by me, with the code being developed by my peers and I.

Throughout this project, I gained valuable skills such soddering, ARDUINO work, pair-programming, RX and TX on a CANLine, and troubleshooting speedometer values.
